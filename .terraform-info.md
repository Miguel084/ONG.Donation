# Nota sobre Infraestrutura (Terraform)

⚠️ **A infraestrutura Terraform foi centralizada no repositório `fcg-infra`**

## 📦 Localização

A configuração Terraform para ONG.Donation está em:
```
fcg-infra/terraform/ong-donation/
```

## 🏗️ Conteúdo

- `aks.tf` - AKS Cluster (Kubernetes)
- `acr.tf` - Azure Container Registry
- `keyvault.tf` - Key Vault com secrets
- `database.tf` - SQL Server + Service Bus
- `variables.tf` - Variáveis parametrizadas
- `outputs.tf` - Valores de saída
- E mais... (veja README em fcg-infra)

## 🚀 Como Usar

### Setup Inicial
```bash
cd ../fcg-infra/terraform/ong-donation
cp terraform.tfvars.example terraform.tfvars
# Editar terraform.tfvars com credenciais Azure
```

### Deploy
```bash
terraform init
terraform plan
terraform apply
```

2. ✅ Push para ACR (Azure Container Registry)
3. ✅ Deploy dos manifestos Kubernetes (k8s/)
